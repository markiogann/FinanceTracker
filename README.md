# FinanceTracker

FinanceTracker — это настольное приложение на C# (Windows Forms, .NET Framework), предназначенное для учёта личных финансов.  
Программа позволяет пользователю добавлять, редактировать и удалять транзакции, управлять категориями расходов и доходов, устанавливать бюджетные лимиты и формировать отчёты в формате PDF.

## Основные функции

- Добавление / редактирование транзакций
- Удаление выбранных операций
- Управление категориями (создание, переименование, удаление)
- Установка лимитов бюджета по категориям или на весь период
- Формирование PDF-отчётов по доходам и расходам
- Автоматическое обновление таблиц и лимитов
- Локальная база данных (SQLite)

## Архитектура проекта

Приложение построено по модульной структуре:

- MainForm — главное окно, отображающее все транзакции и лимиты  
- ManageTransactionsForm — управление транзакциями  
- AddEditTransactionForm — добавление / редактирование операций  
- ManageCategoriesForm — управление категориями  
- ManageBudgetForm — управление лимитами бюджета  
- ReportsForm — формирование отчётов (PDF)  
- DatabaseManager — класс для взаимодействия с базой данных SQLite  

## Технологии

- Язык: C#  
- Среда разработки: Visual Studio 2022  
- Платформа: .NET Framework 4.8  
- База данных: SQLite  
- Средства установки: Inno Setup  

## Как запустить проект

1. Скачайте репозиторий:
   git clone https://github.com/markiogann/FinanceTracker.git
2. Откройте проект в Visual Studio (FinanceTracker.sln)
3. Убедитесь, что в папке ReleaseBuild находится файл базы данных  
4. Нажмите F5 или Запуск без отладки  
5. При первом запуске база данных создаётся автоматически.
